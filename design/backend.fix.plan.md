### 1. Correctness (Math & Logic)

**File:** `src/ucc/backend/backend.cc`

* **Fix Global Phase Loss:** In `lower()`, inside the `OpType::T_GATE` and `OpType::CLIFFORD_PHASE` blocks, the code calculates `bool phase_flip = result.sign ^ op.is_dagger()`. When `phase_flip` is true, we currently emit a dagger gate but fail to accumulate the resulting algebraic global phase ($X T X = e^{i\pi/4} T^\dagger$ and $X S X = i S^\dagger$). When `phase_flip` is true for a `T_GATE`, multiply `ctx.constant_pool.global_weight` by $e^{i\pi/4}$ (e.g., `std::complex<double>(ucc::test::kInvSqrt2, ucc::test::kInvSqrt2)`). For a `CLIFFORD_PHASE` S-gate, multiply it by $i$ (`std::complex<double>(0.0, 1.0)`).
* **Fix Precision Collapse:** In `OpType::NOISE`, change `-std::log(1.0 - std::min(prob_sum, 1.0 - 1e-15))` to use `<cmath>`'s `std::log1p` for numerical stability: `-std::log1p(-std::min(prob_sum, 1.0 - 1e-15))`.
* **Fix Uninitialized Read Risk:** In `lower()`, initialize `uint32_t last_meas_idx = UINT32_MAX;` at the top. Inside the `OpType::CONDITIONAL_PAULI` block, if `op.use_last_outcome()` is true, add `assert(last_meas_idx != UINT32_MAX && "Conditional Pauli executed before any measurement");` before using it.

### 3. Performance & Cache Locality


**Files:** `src/ucc/backend/backend.h` and `src/ucc/backend/backend.cc`

* **Flatten `source_map`:** In `CompiledModule` and `CompilerContext`, change `std::vector<std::vector<uint32_t>> source_map` to a CSR format using two flat vectors: `std::vector<uint32_t> source_map_data` and `std::vector<uint32_t> source_map_offsets`. Update `lower()` to append to these flat arrays, and update Python/Wasm bindings to reconstruct the lists-of-lists dynamically.
* **Hoist $O(N^2)$ Tableau Transpositions:** In `lower()`, `stim::TableauTransposedRaii` is instantiated repeatedly inside the switch cases for `MEASURE` and `CLIFFORD_PHASE`. Hoist this out or restructure the code to avoid toggling the RAII transpose on a per-instruction basis to prevent massive $O(N^2)$ matrix transposes in the tight loop.
* **Eliminate Branches:** Inside `compress_pauli`, replace the tight data-dependent branch: `if (z_bits & (1ULL << q)) { z_bits ^= (1ULL << pivot); }` with the branchless bitwise equivalent: `z_bits ^= (-((z_bits >> q) & 1)) & (1ULL << pivot);`.
* **Fix Telemetry Sync Hazard:** In `lower()`, `ctx.active_k_history.push_back(k_now);` evaluates `k_now` *after* the HIR opcode block finishes. Move this tracking so it captures the correct dimension immediately *before* measurements call `deactivate()`, ensuring pre-measurement SWAP instructions get the correct uncompacted dimension.

### 4. Code Quality & API Polish

**Files:** `src/ucc/backend/backend.cc` and `src/ucc/backend/backend.h`

* **Delete Factory Boilerplate:** Delete all `make_*` factory functions (e.g., `make_frame_cnot`, `make_meas_active_diagonal`). Replace all their invocations in the codebase (including tests) with C++20 designated initializers directly at the `push_back` call sites (e.g., `Instruction{.opcode = Opcode::OP_FRAME_CNOT, .axis_1 = ctrl, .axis_2 = tgt}`).
* **Abstract Algorithmic Duplication:** In `compress_pauli`, the internal bit-clearing `while` loops for Case 1 (X-support) and Case 2 (Pure Z-string) are almost identical. Extract this transversal bit-clearing logic into a shared helper lambda.
* **Extract Duplicated Active-Routing Logic:** In `lower()`, the block that routes a dormant pivot to `next_axis`, appends `H_XZ`, and emits `OP_FRAME_H` + `OP_EXPAND` is duplicated exactly in `OpType::T_GATE` and `OpType::CLIFFORD_PHASE`. Extract this into an inline helper function.
* **Clean Up Unions:** Inside the `Instruction` struct, delete the explicit padding arrays (`uint8_t _pad_a[8];`, `_pad_b[16];`, etc.). The standard inherently pads unions to the largest member (`raw[24]`), and removing these pads allows cleaner aggregate initialization. Ensure we add some static_asserts on the size of Instruction if there are any concerns of this silently breaking.
* **Remove Vacuous Comments:** Delete the low-value comments in `lower()` and `compress_pauli` that restate C++ syntax (e.g., `// Emit diagonal phase gate`, `// Determine classical output index...`, `// Reset reusable scratch tableau to identity for this compression`).
* **Use `std::span`:** Change the signature of `lower()` to accept `std::span<const uint8_t> postselection_mask` instead of a `const std::vector<uint8_t>&`. Make sure to include `<span>` and update `src/python/bindings.cc` to pass data without copying.
* **Enforce `[[nodiscard]]`:** Add the `[[nodiscard]]` attribute to `compress_pauli`, `map_to_virtual`, `lower()`, `trace()`, and `parse()`.
