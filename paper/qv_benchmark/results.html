Analysis

  • QV-20: 1.87x speedup. Fusion reduces 8,967 → 3,089 instructions (65.6%), eliminating ~5,878 redundant array traversals of the 16MB statevector. Each fused OP_ARRAY_U2 replaces 3–5 sequential gate sweeps with a single precomputed 2×2 matrix butterfly.
  • QV-10: 1.54x speedup. Smaller statevector (16KB, fits in L1d) means array passes are cheaper, so the relative gain from eliminating passes is smaller. Dispatch overhead reduction still helps.
  • Cultivation d=5: No change (1.00x). This circuit has no fusible sequences — its 1,518 instructions are dominated by QEC syndrome extraction (CNOT, measurements, noise) with very few consecutive single-axis ops. The pass correctly identifies nothing to fuse.
  • Heuristic validation: The tightened heuristic (require ≥3 array ops, or ≥2 with a rotation) avoids fusing lightweight H+T pairs where the specialized scalar loops outperform a dense 2×2 matrix sweep.
